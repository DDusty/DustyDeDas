Dusty Dassen

Hey, ik ben Dusty Dassen. Een 20 jarige software engineer. Ik volg huidig een opleiding software engineering op het HvA.

Op school leren wij programmeren in java. Hier leren wij bijvoorbeeld OOP in en javaFx. Ook leren wij algorthims and datascructures met java.

De projecten die wij op school maken zijn web based. Dit betekent dat wij voor ons project website's moeten opzetten. Hier staan altijd requirments aan. In het eerste project leerde wij standaard html, css en Jquery. In het 2e project kregen wij wat meer vrijheid. Wij moesten bepalen wat wij gingen gebruiken zolang het maar OOP werd gedaan. Ik en mijn team hebben toen Python met Flask gebruikt. In het 3e project moeten wij gebruik maken van de webframework Angular.

In mijn vrije tijd maak ik graag projectjes om zo mijn programmeer kennis te verbeteren. Projecten waar ik trots op ben laat ik zien in mijn Portfolio → sectie.
